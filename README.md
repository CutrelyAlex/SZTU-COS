# SZTU-COS

完成SZTU-COS评教系统的自动评教，并提供可视化过程展示，仅供测试与学习用途。

## 功能特性

- 自动选择评教选项（除最后一项外全选"同意"，最后一项选"大体同意"）
- 自动填写评教建议（"教学优秀"）
- 实时进度可视化展示
- 智能错误检测和重试机制
- 定时检查空选项提醒
- 支持同一页面多次评教

## 使用方法

### 1. 安装Tampermonkey
首先在浏览器中安装篡改猴扩展程序。

### 2. 导入脚本
1. 下载 `sztu-cos-auto.js` 文件
2. 在Tampermonkey管理面板中点击"添加新脚本"
3. 将脚本内容复制粘贴到编辑器中，或直接拖动该脚本到浏览器中
4. 保存脚本

### 3. 使用脚本
1. 访问深圳技术大学评教系统页面
2. 脚本会自动检测评教页面并开始自动填写
3. 观察页面顶部的进度条了解填写进度
4. 填写完成后会显示成功提示，需要手动点击"完成"和"下一位教师"
